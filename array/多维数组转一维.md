### array


```mysql
 +------------------------------------------------------------
 | 多维数组转一维 关键的是定义一个静态红数组 递归循环就好了
 +------------------------------------------------------------
```

```mysql
function oor($data){
    static $arr = [];
    foreach($data as $k => $v){
        if(is_array($v)){
            oor($v);
        }else{
            $arr[] = $v;
        }
    }
    return $arr;
}
$data = [
    1,['a',],34,
    [233,['o'],241,],
    [65580141, ['i'],],
    ['ah'],
];
$res = oor($data);
print_r($res);
```
